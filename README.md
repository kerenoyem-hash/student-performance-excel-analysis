# Student Performance Excel Analysis

This project demonstrates data entry and summary analysis using Microsoft Excel.

## Project Tasks
- Organized and structured a student performance dataset
- Calculated total scores for each student
- Computed average scores across subjects
- Assigned grades based on average scores
- Created summary tables showing grade distribution by gender
- Built charts to visualize student performance

## Tools Used
- Microsoft Excel

## Dataset Source
https://www.kaggle.com/datasets/mexwell/student-scores
